# fe-yiifan
个人网站重构-前端部分

# 安装
  1. 克隆仓库 `git clone https://github.com/Mutefish0/fe-yiifan`
  2. 安装 `npm install`

# 编译部署 (服务器端)
  1. 拉取更新 `git pull`
  2. 编译 `npm run build`
  3. 部署`public`文件夹即可

# 开发 (本地)
  1. 开启热加载和编译 `npm start`
  2. 浏览器访问`http://localhost:8080`
